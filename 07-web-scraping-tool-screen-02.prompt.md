# PriceWatch: Price History Chart View

## Vibe  
Terminal-style historical price visualization. Dark monitoring interface with ASCII-art inspired Unicode box drawing. Professional pricing analytics dashboard.

## Design System
- **Background**: `#0e0e0f` (dark charcoal)
- **Chart Color**: `#69daff` (cyan blue)
- **Highlights**: `#a1ffc2` (neon green for current price)
- **Border/Grid**: Subtle dark gray lines
- **Text**: White with gray accents

## Page Structure
Display a historical price chart with:
- **Product Title**: At top, minimal styling
- **Y-Axis**: Price scale labels on left side  
- **X-Axis**: Time period labels on bottom
- **Chart Area**: Unicode block characters forming line/area chart
- **Statistics Block Below**:
  - Current Price: Green highlight
  - Lowest Price: Gray text
  - Highest Price: Gray text
  - Average Price: Gray text

## Layout Details
- Left-aligned title and product info
- Centered chart area using Unicode box drawing
- Dark background with minimal borders
- Clean spacing between chart and statistics
- Technical terminal-inspired aesthetic
- No external decorations, focus on data visualization
