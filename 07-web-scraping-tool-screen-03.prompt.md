# PriceWatch: Price Drop Alerts

## Vibe
Alert notifications dashboard highlighting price reductions. Clean, prominent displays of savings opportunities. Modern monitoring interface with emphasis on positive savings amounts.

## Design System
- **Background**: `#0e0e0f` (dark charcoal)
- **Savings Highlight**: `#a1ffc2` (neon green)
- **Original Price**: `#ff7168` (muted red for reference)
- **Text**: White on dark, green accents for savings
- **Status**: Active indicators for alert significance

## Page Structure
Display alert list showing:
- **Product Name**: Left-aligned, primary text
- **Current Price**: Green highlighted, main focus
- **Previous Price**: Muted/struck-through reference
- **Savings Amount**: Large green text, prominent placement
- **Percentage Drop**: Green badge showing discount percentage
- **Time Alert**: When the drop was detected (gray text)

## Layout Details
- Full-width alert list
- Each alert as distinct row/card
- Savings amount in largest text on each row
- Dark background with subtle row separation
- Green accents on all savings values
- Technical, scan-friendly layout
- Organized by recency of price drop
